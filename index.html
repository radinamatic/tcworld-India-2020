Are your PDF documents accessible for users with diverse needs?

PDF A11y testing

PDF accessibility with Acrobat Pro

To fully evaluate PDF files for accessibility you must have access to Adobe Acrobat Pro DC (download the 7 days evaluation version.

PDF Accessibility Checker (PAC 3)

http://www.access-for-all.ch/en/pdf-lab/pdf-accessibility-checker-pac.html
Evaluates PDF files against checkpoints and failure conditions of the Matterhorn Protocol (PDF/UA Conformance Testing Model).

Exercises

Exercise 1: Metadata

Open the PDF file in Acrobat and add the missing title and language.

Download Exercise 1 files (ZIP, 1.4MB)

Exercise 2: Document structure

Open the PDF file in Acrobat, correct the incorrectly nested headings, and the missing list structure tags.

Download Exercise 2 files (ZIP, 800KB)

Exercise 3: Navigation

Open the orignal document and convert to PDF with bookmarks. Open the exported file in Acrobat and correct the link by adding the missing OBJR tag.

Download Exercise 3 files (ZIP, 900KB)

Exercise 4: Images

Open the PDF file in Acrobat, add the missing alternative text to the illustration, and tag the decorative image as an Artifact.

Download Exercise 4 files (ZIP, 900KB)

Exercise 5: Tables

Open the PDF file in Acrobat, add the missing header row tags to the first table, and correct the header scope in the second table.

Download Exercise 5 files (ZIP, 800KB)
